# electricity
# EB Billing & Automated Notification System

This is a JavaFX-based Electricity Billing application with Admin authentication and automated email notification features.

## 🚀 Features
- **Secure Admin Login:** Only authorized personnel can access the billing dashboard.
- **Auto-Billing:** Calculates cost based on units consumed.
- **Email Integration:** Automatically sends the bill summary to the consumer's email.
- **Database Logs:** Stores all billing records in MySQL for future reference.

## 🛠️ Tech Stack
- Java (JDK 17+)
- JavaFX (UI)
- MySQL (Database)
- Jakarta Mail API (Email Service)
